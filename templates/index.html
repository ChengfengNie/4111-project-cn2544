    Welcome to Kaggle

    Kaggle is a data modeling and data analysis competition platform. Businesses and researchers publish data on it, and statisticians and data mining experts compete on it to produce the best models. This crowdsourcing model relies on the fact that there are numerous strategies that can be used to solve almost any predictive modeling problem, and it is impossible for a researcher to know at the outset which method is most effective for a particular problem.

    カグルへようこそ

    Kaggle は、データ モデリングおよびデータ分析のコンペティション プラットフォームです。 企業や研究者はその上でデータを公開し、統計学者やデータ マイニングの専門家は最高のモデルを作成するために競争します。 このクラウドソーシング モデルは、ほぼすべての予測モデリングの問題を解決するために使用できる多数の戦略が存在するという事実に依存しており、研究者が特定の問題に対してどの方法が最も効果的であるかを最初から知ることは不可能です.

    kaggle에 오신 것을 환영합니다

    Kaggle은 데이터 모델링 및 데이터 분석 경쟁 플랫폼입니다. 기업과 연구원은 그것에 대한 데이터를 게시하고 통계학자와 데이터 마이닝 전문가는 최고의 모델을 만들기 위해 경쟁합니다. 이 크라우드소싱 모델은 거의 모든 예측 모델링 문제를 해결하는 데 사용할 수 있는 수많은 전략이 있다는 사실에 의존하고 연구자가 특정 문제에 어떤 방법이 가장 효과적인지 처음부터 아는 것은 불가능합니다. Kaggle의 목표는 크라우드소싱을 통해 이 문제를 해결하여 데이터 과학을 움직이는 것입니다.
